Работа с Node

  1. Получить ноду в переменную

                    $node = Drupal\node\Entity\Node::load(27);

  2. Получить текущую ноду в переменную

                    $node = \Drupal::routeMatch()->getParameter('node');

  3. Проверка на случай если переменная $node является объектом NodeInterface

                    if ($node instanceof \Drupal\node\NodeInterface){}

  4. Получить id текущей ноды

                    $nid = $node->id();

  5. Получить тип текущей ноды

                    $ntype = $node->getType();

  6. Получить значение поля 'field_something' текущей ноди

                    $field_something = $node->get('$field_something')->getValue();

  7. Получить ноду по url

                    $path = \Drupal::service('path.alias_manager')->getPathByAlias('/contacts');
				if(preg_match('/node\/(\d+)/', $path, $matches)) {
